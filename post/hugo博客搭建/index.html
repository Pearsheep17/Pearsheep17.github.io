Hugo博客搭建

1.hugo下载

brew install hugo

2.创建一个hugo文件夹

hugo new site blog

3.选一个喜欢的主题下载

主题网站:https://themes.gohugo.io/
--https://github.com/pravin/hugo-theme-prav--我选择的主题

 git clone https://*gitclone.com*/github.com/... github下载慢加速问题

4.指定主题启动hugo

hugo server -t hyde-hyde --buildDrafts
hyde-hyde是主题文件的名字

5.新建文章

hugo new post/blog.md
post文件夹会自动生成在content文件夹下

6.将文件推到远程仓库

hugo --theme=xxx --baseUrl="..." --buildDrafts
cd public  --进入主题文件夹
git init   --把该文件夹初始化为git仓库
git add .   --把文件加到git仓库
git commit -m "-提交信息-"
git remote add origin https://github.com/pearsheep17/pearsheep17.github.io.git   --关联远端仓库
git push (git push -u origin master)

git pull origin master --allow-unrelated-histories(我第二次换主题的时候用了这个)
