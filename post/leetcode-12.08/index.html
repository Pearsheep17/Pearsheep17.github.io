Leetcode-翻转矩阵后的得分

2020-12-08
leetcode

问题描述：

有一个二维矩阵 A 其中每个元素的值为 0 或 1 。

移动是指选择任一行或列，并转换该行或列中的每一个值：将所有 0 都更改为 1，将所有 1 都更改为 0。

在做出任意次数的移动后，将该矩阵的每一行都按照二进制数来解释，矩阵的得分就是这些数字的总和。

返回尽可能高的分数。

实例：

输入： [[0,0,1,1],[1,0,1,0],[1,1,0,0]]

输出： 39

解释： 转换为 [[1,1,1,1],[1,0,0,1],[1,1,1,1]]

0b1111 + 0b1001 + 0b1111 = 15 + 9 + 15 = 39

提示：

  1. 1 <= A.length <= 20
  2. 1 <= A[0].length <= 20
  3. A[i][j] 是 0 或 1

实现：

假设矩阵为m行n列，由于每一行按照二进制来解释，矩阵得分尽可能高，那么最左边一列的数必须为1。

当将每一行最左边一列翻转为1后，就只能进行列变换了，要使每一列1的数量尽可能多。

  1. 对于最左边的列而言，总贡献为m*2n-1
  2. 对于第j列（j>0），其1的数量最多有k个，那么每一列的总贡献为k*2n-j-1。在统计0，1时要考虑最初进行的行反转

class Solution {
    public int matrixScore(int[][] A) {
        int m=A.length; //行数
        int n=A[0].length;  //列数
        int ret=m*(1<<(n-1));  //第0列总贡献
        for(int j=1;j<n;j++){		
            int cnt1=0;			
            for(int i=0;i<m;i++){
                if(A[i][0]==1){		//第一列未进行行反转
                    cnt1 += A[i][j];
                }else{		//第一列进行了行反转
                    cnt1 +=(1-A[i][j]);
                }
            }
            int k=Math.max(cnt1,m-cnt1);	//选择1和0的大者赋给k
            ret +=k*(1<<(n-j-1));		
        }
        return ret;
    }
}
